## 🤖 AI Projects

### [CMA Editor](https://github.com/Heatescape/cma_editor)
> AI-assisted CMA report generator for real estate professionals

A full-stack web app that automates the most time-consuming parts of producing Comparative Market Analysis reports. It scrapes comparable sales data directly from REA, extracts data from Cotality PDF uploads, auto-generates Google Maps tiles, and produces a polished PDF — all from a browser UI with no code required.

**Stack:** `Python` `FastAPI` `PyMuPDF` `ReportLab` `Playwright` `Google Maps API`  
**Key features:** REA web scraping · Cotality PDF parsing · AI-positioned map markers · Drag-and-drop comparable management · One-click PDF export

---

### [VPI Facebook Bot](https://github.com/Heatescape/vpi_facebookbot)
> GPT-powered real estate outreach automation

A Flask dashboard that helps real estate agents reconnect with past clients via Facebook. It syncs unconditional clients from Portal Plus, scrapes Facebook profiles with Playwright, and uses GPT-4o-mini to generate personalised introduction messages in Chinese, English, or Korean.

**Stack:** `Python` `Flask` `OpenAI API` `Playwright` `SQLite` `pandas`  
**Key features:** Portal Plus one-click sync · AI-personalised multilingual messages · Contact deduplication · Full message history archive
